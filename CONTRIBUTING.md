# 🤝 Contributing to Agape

We are building a sanctuary, not a product. To contribute, you must adhere to the **Trinity Protocol**:

1. **Non-Invasive Logic:** Code that attempts to predict individual human behavior for the purpose of extraction will be rejected.
2. **Modular Sovereignty:** Ensure all features can be "forked" at the individual level.
3. **The Verify-First Rule:** Any new data-ingestion logic must include a "Perspective Range" hook to prevent single-source bias.

**How to start:** Check the `IDEAS.md` for the current "Verified Library" roadmap.
